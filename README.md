# Вариант 12 - Гостиничные номера

## Описание
Программа для управления данными о гостиничных номерах.

## Функции
- Вывод всех 20 номеров
- Фильтрация номеров, название отеля которых оканчивается на "house"
- Сортировка отфильтрованных номеров по вместимости (пузырьком)
- Поиск номера по номеру комнаты
- Вывод 3 самых дорогих люксов
- Фильтрация номеров с ванной
- Изменение данных номера

## Структуры
- `Comfort` - перечисление (люкс, полулюкс, стандарт, эконом)
- `Amenities` - бытовые опции (Wi-Fi, TV, ванна, балкон, мини-бар)
- `HotelRoom` - гостиничный номер

## Требования задания
- ✅ Функции для каждой подзадачи
- ✅ Поле с enum
- ✅ Вложенная структура
- ✅ Инициализация 20 записей в коде
- ✅ Сортировка пузырьком




# Отель: работа с номерами и файлами в C++

## Описание проекта

Программа для управления гостиничными номерами с возможностью:

- Хранения информации об отелях, номерах, их комфортности, стоимости и удобствах
- Фильтрации и сортировки номеров
- Обновления цен из текстового файла
- Сохранения и загрузки данных в бинарные файлы

## Задание 9 (основа)

Реализована структура данных для гостиничных номеров:

- **HotelRoom** - содержит название отеля, номер комнаты, комфортность, вместимость, цену и удобства
- **Amenities** - набор удобств (Wi-Fi, TV, ванна, балкон, мини-бар)
- Фильтрация отелей, оканчивающихся на "House"
- Сортировка по вместимости
- Поиск номера по номеру
- Топ-3 самых дорогих люксовых номера
- Фильтрация номеров с ванной

## Задание 10 (работа с файлами)

### 1. Чтение из текстового файла

Функция `update_prices_from_file()` читает из текстового файла пары `номер_комнаты` и `новая_цена`, после чего обновляет соответствующие записи в массиве.

**Формат файла `price_update.txt`:**
```
101 27000
105 38000
112 5200
```

### 2. Бинарные файлы

- **`save_to_binary()`** - запись структуры `HotelRoom` в бинарный файл с корректной обработкой строк (сначала длина, затем содержимое)
- **`read_from_binary()`** - чтение структуры из бинарного файла с возможностью выбора записи по позиции

**Особенности реализации:**
- Использование `ios::binary` для бинарного режима
- Использование `ios::app` для добавления записей в конец файла
- Корректная работа с `string` (запись длины + данных, а не указателя)

## Технологии

- C++ (стандарт C++11 и выше)
- Библиотека `fstream` для работы с файлами
- Библиотека `string` для работы со строками
